# Pathfinder

**Pathfinder** is a web application designed for runners, walkers, and hikers to find AI-generated and personalized route suggestions based on their location. The project aims to improve route planning by incorporating AI-based recommendations in future updates.

---

## 🚀 Features
- 🏃 **AI-Generated Running Routes** *(In Progress)*
- 🌍 **Location-Based Route Suggestions**
- 📍 **Personalized Route Creation**
- 📊 **Database-Backed Storage for User Routes**
- 🌐 **Flask-Based Web Framework**
- 🗺️ **Integration with Mapping APIs (Planned)**
